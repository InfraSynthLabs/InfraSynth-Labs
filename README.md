# InfraSynth Labs

InfraSynth Labs is a personal infrastructure and security lab focused on practical systems administration, networking, firewalling, virtualization, documentation, and self-hosted services.

This repository contains the public website and documentation for the lab.

## Purpose

The goal of InfraSynth Labs is to document real infrastructure work as it is designed, built, tested, broken, fixed, and improved.

This project is intended to show hands-on experience with:

- Network infrastructure
- Firewall deployment
- DNS and DHCP
- Cisco switching
- Virtualization
- Linux administration
- Cloudflare Pages
- Public documentation
- Troubleshooting and operational notes
